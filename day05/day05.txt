
 
 디스플레이
    display
        block
            한 줄 전체 너비를 차지
            크기를 원하는 대로 지정 가능
            (div, p, 시멘틱 웹태그, h1~h6 ...)

        inline
            이웃하는 요소와 나란히 배치
            크기를 원하는 대로 지정이 불가능, 안에 있는 컨텐츠의 크기만큼 크기가 지정
            (a, span ...)

        inline-block 
            이웃하는 요소와 나란히 배치, 크기 지정 가능 
            inline과 block의 속성을 모두 갖고 있는 것

        none
            보이지 않는 상태
            사용자 화면에서 완전히 사라짐
 
 포지션
    position (위치 지정 방식)
        static: 기본 값
        relative : 지정 요소 (static + 내 자신이 기준)
                 : absolute와 함께 쓰이는 경우가 많고
                 : 내 현재 위치에 상대적으로 top, left, right, bottom 이동이 가능
        
        absolute : 가장 가까운 부모 요소 중 지정 요소에 상대적으로 배치
                   절대 위치 지정 방식
                   다른 요소와 상관없이 겹쳐짐

        fixed : 절대 위치 지정 방식
                뷰포트(사용자 화면)에 따라 스크롤과 상관없이 고정되어 배치
                다른 요소와 겹치는 속성 때문에 특정 요소가 가려질 수 있음
                만약 의도한 바가 가려지는 것이 아니라면 padding을 이용해서 여백을 조정

        * sticky : fixed + relative, 가장 가까운 스크롤이 되는 조상과 
                   표 관련 요소 기준으로 지정한 값이 넘으면 마치 fixed처럼 고정
